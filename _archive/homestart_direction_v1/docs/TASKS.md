If your team has only 3 people: collapse roles 2 and 4. The Strategist becomes co-presenter.

## What to do if you're behind

**Plan B if Designer is behind at T+5:00 on Screen 3:** Designer focuses on Screens 1 and 2 only; Screen 3 becomes a *list* of components rendered as a wireframe sketch screenshot inside the slide, with the verbal pitch carrying the moat. This is uglier but still defensible.

**Plan C if the deck isn't done at T+11:00:** Cut Slide 12 (roadmap can be spoken in 15 seconds against an empty title slide). Cut Slide 14 appendix (Q&A handled verbally).

**Plan D — bare minimum to submit:** Slides 1, 2, 3, 6, 7, 10, 13. The reframe + the solution + the demo + the ask. That is the irreducible core. Everything else is nice-to-have.

---

## What NOT to do

- **Do not redesign the reframe at 3 AM.** If someone has a "what if we…" insight after T+5:00, write it on the parking-lot list in Discord and ship the current plan.
- **Do not skip dry runs to keep building.** The deck does not need to be perfect; the *delivery* needs to be smooth. Practice beats polish.
- **Do not let any one person work in isolation for more than 90 minutes.** The 90-min status checkpoint is mandatory.
- **Do not change the product name after T+0:40.** Bikeshedding kills hackathons.
- **Do not pull all-nighters.** A team that's slept beats a team with a slightly prettier deck every time.
