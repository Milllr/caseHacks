## The thesis

The case asks whether 18-to-34 Canadians lack confidence or capital. The data says the question is wrong. The cohort is bimodal, and the real gap is that **Scotia is absent from the moment of investment decision**—while its only structural advantage (transaction-data visibility) has a known expiration date inside the 18-month case horizon when Open Banking Phase 1 lands.

Scotia HomeStart targets ONE segment (Capital-Constrained Curious — ~58% of cohort), at ONE moment (tax-time + FHSA, Feb–Mar 2027), using ONE structural asset (the FHSA → Scotia mortgage closed loop). Wealthsimple structurally cannot replicate that loop inside the horizon.
